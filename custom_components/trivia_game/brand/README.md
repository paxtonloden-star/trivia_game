# Brand images

Home Assistant custom integrations can ship their own brand images directly in this folder.

Add these files here:

- `icon.png`
- `logo.png`

Optional dark variants:

- `dark_icon.png`
- `dark_logo.png`

Optional high-resolution variants:

- `icon@2x.png`
- `logo@2x.png`
- `dark_icon@2x.png`
- `dark_logo@2x.png`
